Frequently asked question

What is the Document Object Model (DOM)?

The (Document Object Model) is a programming artboard for web documents , It represents the structure of a web page as a tree of objects. It allow us to maake changes in the documents structure or style. It mainly connect html css with the javascript !

How do you select an element from the DOM?

We can select an element from the DOM in many ways but most common ways are getElementById which gets the element by their id and getElementsByClassName which target the class name . There are other ways are available aswell such as getElementsByTagName and querySelector

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where a single event listener is added to a parent element to manage events for its all child element . improving performance and handling dynamic content efficiency.

How do you manipulate an element's attributes and styles using the DOM?

There are many way to manipulate an element such as we can use getAttribute and setAttribute, also we can use element.style to add style tags !
